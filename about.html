Nedir?

UserLooperGen, kullanıcıların kendi loop müziklerini kolayca oluşturmasını sağlayan ücretsiz bir web uygulamasıdır. EnderArcane Studio tarafından geliştirilmektedir.

Vizyon

Profesyonel müzik ekipmanlarına veya pahalı yazılımlara ihtiyaç duymadan, yalnızca bir tarayıcı ile müzik üretebilmek.

Özellikler

4x4 pad sistemi — 16 adet ses yuvası
MP3 ve WAV ses desteği
Müzik kaydı ve dışa aktarma
.ulbm proje dosyası formatı
Tamamen offline çalışır
Mobil uyumlu tasarım

Teknoloji

Tamamen HTML, CSS ve JavaScript ile geliştirilmiştir. Web Audio API kullanır. Hiçbir backend sunucusu yoktur — her şey cihazında çalışır.
